# ERP Future Plan

## Overview

This folder contains the future plan for ERP enhancements and AI-powered features.

## Files

| File | Description |
|------|-------------|
| [ai-features.md](./ai-features.md) | AI suggestions for users (assistant, smart search, recommendations) |
| [erp-enhancements.md](./erp-enhancements.md) | General ERP power-ups (workflows, bulk actions, analytics) |

## Suggested Phases

| Phase | Focus | Effort |
|-------|--------|--------|
| **1** | Validation warnings, contextual help | Low |
| **2** | Bulk actions, audit trail, keyboard shortcuts | Medium |
| **3** | Smart search, AI suggestions | Medium |
| **4** | Approval workflows, dashboards, reporting | High |
| **5** | AI assistant, natural-language queries | High |

## Updating

- Add new ideas to the relevant file
- Prioritize by effort vs impact
- Review quarterly and adjust phases
